Phase 1 : Awareness & exposure (build up hands)

"How many of you have heard about Git?" → raise one hand

"How many of you have used Git, at least once?" → raise one hand

0 hands = total beginners
1 hand = awareness only
2 hands = some experience

Phase 2 : Barriers (keep it inclusive)

(Everyone can participate, regardless of experience)

“How many of you haven’t used Git because you never really needed it (yet)?” → raise a hand 

Now, just smile if you: 

“How many of you haven’t used Git because it felt a bit intimidating to get started?” 


Phase 3 : Experience breakdown (now hands go down)

Say: “If you have two hands up, this part is for you.”

“How many of you have created a repository yourself?” → put down one hand
“How many of you have worked on a shared repository?” → put down your other hand


2 hands up = used Git but not independently
1 hand up = solo user
0 hands up = collaborative user


Say, we hope the lesson of today help you to know and use Git more confidently in your work.
